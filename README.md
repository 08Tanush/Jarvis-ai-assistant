# Jarvis AI Assistant 🤖🎤

A simple voice-controlled AI assistant built in Python using `speech_recognition`, `pyttsx3`, and `webbrowser`. Jarvis listens to voice commands and can:

- 🗣️ Recognize and respond to voice input
- 🌐 Open websites like Google, YouTube, Spotify, and more
- 🌦️ Fetch real-time weather data using OpenWeatherMap API
- 📝 Remember and recall notes
- 🕒 Tell the current time

## Features

- Voice-based interaction using `speech_recognition`
- Text-to-speech replies with `pyttsx3`
- Weather updates via OpenWeatherMap API
- Smart website launcher
- Basic memory system using `.txt` file
- Clean Python script with error handling

## Requirements

- Python 3.x
- Libraries: `speech_recognition`, `pyttsx3`, `requests`, `datetime`, `webbrowser`

Install them using:

```bash
pip install speechrecognition pyttsx3 requests
